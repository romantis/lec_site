# Front End for Language Exchange Club

## Prerequisites
* [Node 20](https://nodejs.org/en/) or greater.

## Installation
```bash
 git clone https://github.com/romantis/lec_site.git
 cd lec_site
 npm install
```

## Development
```bash
npm start
```
[http://localhost:3000/](http://localhost:3000/)


## Build
```bash
npm run build
```
